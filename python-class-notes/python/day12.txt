		String Handling
		----------------
what is String?
---------------
a string is a charecter/collection of charecters/sequence of words.

in python, the string datatype having 'str' class

in python, the string object is a sequence,iterable and immutable object.

how to represent the string objects in python?
------------------------------------------------
we can represent the string objects in python, in that case we are using following options,

	1).single line string representations

		a). by using single-single quote's
			ex: 'siva'

		b). by using single-double quote's
			ex: "siva"

	2).multi-line string representations
		
		a). by using tripple-single quote's
			ex: '''siva'''

		b). by using tripple-double quote's
			ex: """siva"""


ex1:
----
	demo.py
	-------

y="krishna"
z='''siva
ameerpet
