if elif else statement
------------------------
	syntax
	------
	if condition:
		stmt_1
		stmt_2
		......
		stmt_n
	elif condition:
		stmt_1
		stmt_2
		......
		stmt_n
	else:
	    stmt_1
	    stmt_2
            .....
	    stmt_n

flowchart:
----------
	ref fig: if elif else flowchart

ex1:
----
write a python program to check whether given integer is either positive or negative or zero integer?

	demo.py
	------
num=int(input("enter your number: "))
if num>0:
    print("Positive Integer")
elif num<0:
    print("Negative Integer")
else:
    print("Zero Integer")

	(or)

num=int(input("enter your number: "))
if num>0:
    print("Positive Integer")
else:
    if num<0:
        print("Negative Integer")
    else:
        print("Zero Integer")

	(or)

num=int(input("enter your number: "))
print("Positive Integer" if num>0 else "Negative Integer" if num<0 else "Zero Integer")

output1:
--------
enter your number: 4
Positive Integer

output2:
--------
enter your number: -2
Negative Integer

output3:
--------
enter your number: 0
Zero Integer

elif ladder:
------------
the elif ladder means to define single if with multiple elif blocks.

in elif ladder, the else block is a optional.

	syntax
	------
	if condition:
		stmt_1
		stmt_2
		......
		stmt_n
	elif condition:
		stmt_1
		stmt_2
		......
		stmt_n
	elif condition:
		stmt_1
		stmt_2
		......
		stmt_n
	elif condition:
		stmt_1
		stmt_2
		......
		stmt_n
	------
	------
	------
	else:
	    stmt_1
	    stmt_2
	    ......
	    stmt_n

eval()
------
to read one or more input values from the user input with homogenious and hetrogenious.

in eval() function, our input values are must be seperated by "," only.
