Advantages of Python
--------------------
1). Simple Syntax

	in python, there's no concept of ;,(),{}
	python simply follows space indentation concept

ex: if block representation
---

	C/C++/Java			Python
	----------			-------
	if(condition)			if condition:
	{					stmt_1
	   stmt_1;				stmt_2
	     stmt_2;				......
	   ......				stmt_n
	 stmt_n;
	}


2). Less Code

	python programs having Less Code when compared with other programming Languages programs code like C,C++,Java,....

ex:
---
write a C program to print "hello world"?

	demo.c
	------
#include<stdio.h>
void main()
{
    printf("hello world");
}

write a Java program to print "hello world"?

	demo.java
	---------
import java.io.*;
class Test
{
    public static void main(String[] args)
    {
        System.out.println("hello world");
    }
        
}

write a Python program to print "hello world"?

	demo.py
	--------
print("hello world")

3). Dynamically Typed Language

ex:
---
write a C program(Static Typed Language) to perform the addition operation on two different types of numerical values?

	demo.c
	------
#include<stdio.h>
void main()
{
    int x=10;
    float y=2.3;
    float z=x+y;
    printf("%f",z);
}


write a Python program(Dynamically Typed Language) to perform the addition operation on two different types of numerical values?

	demo.py
	-------
x=10
y=2.3
z=x+y
print(z)
