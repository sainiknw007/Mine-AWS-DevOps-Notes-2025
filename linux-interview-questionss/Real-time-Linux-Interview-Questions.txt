What is cron and at command what is difference between them ?
Purpose: cron job
Schedule repetitive tasks at fixed intervals (daily, weekly, monthly, etc.).

* * * * * command_to_run
| | | | |
| | | | ----- Day of week (0-7) (Sunday=0 or 7)
| | | ------- Month (1-12)
| | --------- Day of month (1-31)
| ----------- Hour (0-23)
------------- Minute (0-59)

2. at Command

Purpose: Schedule a one-time task at a specific time.

How it works: Uses the atd daemon to run the command once at the specified time.

Usage:

at 14:30
at> /home/user/backup.sh
at> <Ctrl+D>   # To save
